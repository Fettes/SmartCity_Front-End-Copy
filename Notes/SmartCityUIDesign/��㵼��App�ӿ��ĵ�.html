接口域名：http://trash.jzbwlkj.com
请求方式 : POST

1.添加任务

 url  /api/trashcan/update_task

 参数 

 task_id 任务id

 返回

2.获取车辆信息

 url /api/Trashcan/getcars

 参数

number  车牌号，不传参数返回全部汽车信息

返回

  "message": "操作成功！",

    }

  ]

}
