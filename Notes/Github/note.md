# Init project

```
// clone from your repo link at first time
git clone xxx 

<!-- commit to github -->

<!-- fetch from github -->
git pull

<!-- add file into index -->
git add .

<!-- commit to local repro -->
git commit -m "some message" --no-verify

<!-- push to github -->
git push

```

# How to run 

```
// install dependencies at first time
npm install
// run your project
npm run dev

```
